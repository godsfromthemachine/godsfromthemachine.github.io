Every project in the Gods from the Machine pantheon is a specialized, autonomous entity. Each one is designed to run entirely on local hardware using small, capable AI models &mdash; no cloud, no API keys, no subscriptions.

## Naming Convention

Each project is named after a god or legend from world mythology, reflecting its specialized role in the ecosystem:

- **Gilgamesh** &mdash; the ancient Sumerian king who sought immortal knowledge. A coding agent that writes, tests, and improves code autonomously.
- **Zeus** &mdash; king of the Greek gods, ruler of lightning. A build system that orchestrates local AI model serving.
- **Raijin** &mdash; Japanese god of thunder and lightning. Lightning-fast inference execution.
- **Garuda** &mdash; divine bird vehicle of Vishnu in Hindu mythology. Soars over directory trees, providing a bird's-eye view.
